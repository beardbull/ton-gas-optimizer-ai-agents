# ⚡ TON Gas Optimizer + AI

AI-powered tool for optimizing gas fees on TON blockchain.

## ✨ Features
- 🌓 Dark/Light theme with smooth animations
- 💰 Real-time TON/USD price (CoinGecko)
- 🔐 Mainnet & Testnet support
- 🤖 AI-powered gas optimization
- 📊 DeFi router & contract analyzer

## 🚀 Quick Start
1. Get API key at https://toncenter.com
2. Run: `streamlit run demo/app.py`
3. Enter API key in sidebar
4. Connect wallet & optimize!

## 🔑 API Setup
- Testnet: works without key
- Mainnet: requires free API key from @toncenter_bot

## 📦 Requirements
- Python 3.11+
- pip install -r requirements.txt
